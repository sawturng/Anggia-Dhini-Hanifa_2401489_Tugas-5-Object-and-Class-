# Pemrograman Berorientasi Objek — Object dan Class Java

## Deskripsi
Program ini merupakan implementasi konsep Object dan Class dalam Java. Program digunakan untuk menghitung luas dan keliling segitiga menggunakan pendekatan OOP.

---

## Konsep yang Digunakan
- Class dan Object  
- Atribut  
- Method  
- Konstruktor  
- Pemanggilan method  

---

## Struktur Program

### Class Segitiga
- Atribut: alas, tinggi, sisi  
- Method:
  - getLuas() → menghitung luas  
  - getKeliling() → menghitung keliling  
- Konstruktor untuk mengisi nilai awal  

### Class Object_Class
- Class utama (main)  
- Membuat objek segitiga  
- Menampilkan hasil perhitungan  

---

## Cara Kerja
1. Membuat objek segitiga  
2. Menghitung luas dan keliling  
3. Menampilkan hasil ke layar
